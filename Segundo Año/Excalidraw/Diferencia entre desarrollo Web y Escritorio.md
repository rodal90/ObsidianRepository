4) Diferencia entre ORM y SQL: 

SQL (Structured Query Language)

Definición: SQL es un lenguaje de programación estándar utilizado para gestionar y manipular bases de datos relacionales.

Uso: Se utiliza para realizar consultas directas a la base de datos, como seleccionar, insertar, actualizar y eliminar datos.
Ventajas:

Control total: Permite un control detallado sobre las consultas y optimizaciones.

Rendimiento: Puede ser más eficiente en términos de rendimiento si las consultas están bien optimizadas.
Desventajas:

Complejidad: Puede ser complicado y tedioso escribir y mantener consultas SQL complejas.

Portabilidad: Las consultas SQL pueden no ser portables entre diferentes sistemas de gestión de bases de datos (DBMS).
ORM (Object-Relational Mapping).

Definición: ORM es una técnica de programación que permite convertir datos entre sistemas incompatibles usando lenguajes de programación orientados a objetos.

Uso: Facilita la interacción con la base de datos mediante la creación de objetos en el código que representan las tablas de la base de datos.

Ventajas:

Abstracción: Simplifica el acceso a la base de datos al abstraer las consultas SQL.

Productividad: Aumenta la productividad del desarrollador al reducir la cantidad de código SQL que necesita escribir.

Mantenimiento: Facilita el mantenimiento del código al trabajar con objetos en lugar de consultas SQL.

Desventajas:

Rendimiento: Puede ser menos eficiente en términos de rendimiento comparado con SQL puro, especialmente en consultas complejas.

Flexibilidad: Puede limitar el control sobre las consultas y optimizaciones específicas de la base de datos.

En resumen, SQL ofrece un control más detallado y puede ser más eficiente, pero puede ser más complejo de manejar. ^zf60bxS6

5) JSP, PHP:     

JSP (JavaServer Pages)

Definición: JSP es una tecnología que permite a los desarrolladores crear páginas web dinámicas utilizando el lenguaje de programación Java. Las páginas JSP contienen código Java incrustado en HTML, que se ejecuta en el servidor para generar contenido dinámico1.
Uso: Se utiliza principalmente en aplicaciones web basadas en Java, permitiendo la separación de la lógica de negocio y la presentación.

Ventajas:

Portabilidad: Al estar basado en Java, es multiplataforma.
Integración: Se integra bien con otras tecnologías Java como servlets y frameworks como Spring.


PHP (Hypertext Preprocessor)

Definición: PHP es un lenguaje de programación de código abierto, especialmente adecuado para el desarrollo web. PHP se incrusta en el HTML y se ejecuta en el servidor, generando contenido dinámico que se envía al navegador del cliente2.
Uso: Es ampliamente utilizado para desarrollar sitios web y aplicaciones web, desde blogs hasta sistemas de gestión de contenido.

Ventajas:

Facilidad de uso: Es fácil de aprender y usar, lo que lo hace popular entre los desarrolladores.
Flexibilidad: Puede integrarse con una amplia variedad de bases de datos y servicios web.
